Local References

You can use markers in your HTML template (called Local References) that you can use to obtain and use the specific element that can be used anywhere in the HTML template. You can use these local references in any HTML element by placing a # followed by the name of the reference inside the opening tag of the element.

Example:

< p #localRefName >< /p >

Once you place a local reference inside an element, you now have access to use the element and all of its attributes inside of the HTML template. You can see this in action by pressing the button and viewing our local referenced paragraph in our template in the browser console.

If you want access to a method before you call a method in the HTML template you can use the @ViewChild decorator.

This element can be accessed in the TS file and manipulated whenever we want to.
