Routing

app.module.ts

To start with routing, we begin in the app.module.ts file to import and configure our routes.

We start by importing the Routes, RouterModule from @angular/router.

After we import Routes, we now have access to the Routes Type which we can use when we start creating our routes. We creats an appRoutes const (still in the app.module.ts file) and make it an array with the array items being of type Routes.

We can now fill out this array with our routes, which will be individual objects giving information that will point to the route.

{ path: 'routing', component: RoutingComponent }

As you can see we give the object a path property - which will be the location of the route - www.mywebsite.com/routing. This is followed by the component that you want displayed when the user navigates to this route.

Route Navigation

Instead of using < a > tags to navigate through your application, (a link tags refresh your webpage every time you click one) Angular provides the routerLink attribute which you can apply on any HTML element and you provide the desired destination and when clicked, the route will change to the component you specified in the app.module.ts file.
